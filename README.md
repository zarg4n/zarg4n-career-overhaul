# zarg4n Career Overhaul

EA SPORTS FC 26 TU1.6.4 için kariyer modunu daha canlı, daha tutarlı ve daha futbol odaklı hale getiren bağımsız bir proje.

Author / Yapımcı: **zarg4n**

Repository: [github.com/zarg4n](https://github.com/zarg4n)

## Türkçe

Bu proje KIARIKA veya başka bir modun dosyalarını kullanmaz. Amaç, iyi kariyer modu fikirlerini kendi tablolarımız, kendi runtime kodumuz ve kendi diyaloglarımızla yeniden kurmaktır.

İlk geliştirilen çekirdek:

- Genç oyuncular için gizli gelişim profilleri
- Dakika, maç reytingi ve sezon istatistiklerine dayalı gelişim
- Gol/asist gibi kısa süreli katkıların kaybolmaması
- Sezon sonunda sınırlı potansiyel güncellemesi
- Yaşa bağlı ve düşük ihtimalli boy/kilo gelişimi
- Bruiser ve diğer PlayStyle adaylarının fiziksel/mental profille ilişkilendirilmesi
- 80/85 OVR çevresinde kontrollü PlayStyle+ adayları
- TU1.6.4 için yeni kariyer zorunluluğu
- Gameplay fiziğine dokunmayan kariyer odaklı mimari

Planlanan devam modülleri:

- Oyuncu hafızası, kişilik ve takım ilişkileri
- Maç sonrası oyuncu mesajları
- Maç olaylarına duyarlı basın toplantıları
- Oyuncu/klüp transfer görüşmeleri ve blöf sistemi
- Türkçe ve İngilizce kariyer diyalogları
- Oyuncu detayları ve gelişim planı için kariyer HUD'si
- Genişletilmiş isim havuzları
- Create-a-Club kadro oluşturma ekranı

## English

An independent career-mode project for EA SPORTS FC 26 TU1.6.4.

The project does not use KIARIKA or any other mod’s files. It rebuilds the intended systems with zarg4n-owned data, runtime code and dialogue content.

The first core slice includes:

- Hidden development profiles for youth players
- Development based on minutes, match rating and season statistics
- Short cameo performances still matter without causing runaway growth
- Limited end-of-season potential adjustments
- Age-gated, low-probability height and weight development
- Physical and mental profile links for Bruiser and other PlayStyle candidates
- Controlled PlayStyle+ candidate generation around 80/85 OVR
- New-career requirement for TU1.6.4
- Career-focused changes without gameplay physics edits

Planned modules:

- Player memory, personality and squad relationships
- Post-match player conversations
- Match-aware press conferences
- Player/club transfer negotiations and bluffing
- Turkish and English career dialogue
- Career HUD panels for player details and development plans
- Expanded name pools
- Create-a-Club squad creation tools

## Package

The project uses one static `.fifamod` package and one Live Editor Lua runtime. The static package is for database/UI assets. The Lua runtime is required for dynamic state, season processing and career events.

The current repository is the source and runtime development package. The compiled `.fifamod` is produced with FIFA Editing Toolsuite for the exact FC 26 TU1.6.4 installation.
