# zarg4n Career Overhaul 0.1.0-dev

## Türkçe

Bu, zarg4n Career Overhaul’un ilk geliştirici sürümüdür. TU1.6.4 için bağımsız kariyer runtime’ı, genç oyuncu profilleri ve gelişim hesaplamasının temelini içerir. Final `.fifamod` paketi için FIFA Editing Toolsuite derlemesi gerekir.

## English

This is the first developer release of zarg4n Career Overhaul. It contains the TU1.6.4-independent career runtime foundation, youth-player profiles and the first development calculations. The final `.fifamod` package must be built with FIFA Editing Toolsuite.
