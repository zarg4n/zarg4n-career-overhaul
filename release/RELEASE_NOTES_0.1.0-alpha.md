# zarg4n Career Overhaul 0.1.0-alpha

## Türkçe

İlk oynanabilir çekirdek sürüm. Statik genç oyuncu üretim ayarı ve save’e özel Live Editor gelişim sistemi birlikte gelir. Potansiyel artışı yalnızca yeterli sezon örneği olan güçlü performanslarda devreye girer; az maçlık iyi bir örnek gelişime küçük katkı yapar fakat tek maç potansiyeli değiştirmez. Fizik ve PlayStyle sistemi kontrollü eşiklerle ilerler. Gameplay verisi değiştirilmez.

Bu alpha sürümünde kariyer HUD’si, yeni konuşmalar, isim havuzu, Create-a-Club seçici, transfer rol yapma sistemi ve pozisyon uyarısı düzeltmesi bulunmaz.

## English

The first playable core release. It combines conservative youth-generation tuning with save-specific Live Editor development. Potential only changes after a meaningful seasonal sample; a strong small sample provides a modest development reward but one match cannot change potential. Physical growth and PlayStyle awards use controlled thresholds. Gameplay data is untouched.

The career HUD, new conversations, expanded name pools, Create-a-Club selector, transfer role-play and position-warning fix are not part of this alpha.
